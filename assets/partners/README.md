# Logos de Parceiros

Coloque aqui os logos reais das empresas atendidas pela marcenaria.

A landing page monta os caminhos automaticamente com base no nome visivel do parceiro.

Exemplos esperados hoje:

- `casa-prime.svg`
- `studio-forma.svg`
- `grand-office.svg`
- `villa-design.svg`
- `urban-interiors.svg`
- `maison-concept.svg`
- `atelie-nova.svg`
- `linea-corporate.svg`

Voce pode usar `.svg`, `.png`, `.jpg` ou `.webp`, mas se mudar a extensao,
atualize a logica em:

- `script.js`

Recomendacoes:

- fundo transparente quando possivel
- logos horizontais
- largura entre `300px` e `800px`
- boa leitura sobre fundo claro
